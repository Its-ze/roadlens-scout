ESP32 sensor flasher and Android field app

GitHub

Flash the sensor. Load the map.

Auto-detected ESP32-family sensor firmware, Android APK, checksums, and release metadata in one Pages-ready package. This firmware advertises over BLE first, then starts passive detection after the phone connects. The Android app can also run a phone-side BLE sweep and update OTA-capable sensors over Wi-Fi.

Download APK Open Flasher Page
Auto chip detection Installer picks ESP32, ESP32-S3, or ESP32-C3 builds automatically.
Wider Wi-Fi sweep Firmware scans channels 1-11, reports counters, and matches the current 42-prefix set.
Phone USB setup The APK Setup tab detects a plugged-in ESP32 and opens the matching RoadLens flasher.
Sensor OTA After one USB/web flash, the app can detect and update outdated sensor firmware over Wi-Fi.
Phone BLE sweep The APK can scan BLE names, manufacturer IDs, and known prefixes from the phone.
